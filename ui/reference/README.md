# UI Reference Workspace

This folder is reserved for generated HTML or Stitch reference files used as visual input for the AJ Digital OS local UI shell.

Rules:

- treat reference files as design/layout inputs only
- do not ship raw generated reference markup as production runtime code
- prefer semantic tokens and reusable components in `src/ui/` over duplicated inline styles or placeholder labels

Expected reference themes:

- command center / dashboard
- chat + deliverables
- settings / API credentials / model profiles
- integrations / tool registry
