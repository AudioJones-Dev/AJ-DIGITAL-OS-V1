# Security Policy

AJ Digital OS handles approval-gated AI workflows, webhook traffic, and credential references. We take security reports seriously.

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security reports.

Email the maintainers privately or open a private security advisory via GitHub.

You should expect an acknowledgment within 5 business days.

## Scope and policy details

For full details — disclosure process, secret-handling policy, runtime-state policy, dependency scanning, and operator hygiene expectations — see [`docs/SECURITY.md`](docs/SECURITY.md).
