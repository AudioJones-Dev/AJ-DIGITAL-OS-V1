# AJ Digital OS v0.1.0

Baseline release of AJ Digital OS.

## Included in this release
- Installable CLI entrypoint
- Operator command layer
- Approval and execution lifecycle
- Observability services and terminal inspection commands
- Operator, recovery, architecture, onboarding, and publish-preparation documentation

## Notes
This is an early `0.1.0` baseline release intended for internal and controlled usage.
Interfaces may still evolve during the `0.x` phase.
