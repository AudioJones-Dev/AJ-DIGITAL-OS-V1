# Brand Memory

## Purpose

Stores approved brand voice, positioning, naming, copy constraints, and offer language for AJ Digital OS and related Audio Jones properties.

## What Belongs Here

- Brand voice rules
- Offer naming and positioning
- Approved vocabulary
- Copy constraints
- Public-facing messaging principles

## What Does Not Belong Here

- Draft copy not approved for reuse
- Client-private strategy unless tenant-scoped
- Secrets or account data
- Legal advice or compliance claims without source review

## Write Access

Human operators approve brand memory. Agents may draft changes but should not alter public positioning without approval.

## Agent Read Access

Agents may read brand memory through the Memory Router when producing copy, docs, UI text, or public-facing artifacts.

## Promotion Path

Approved brand facts become canonical `brand_memory` and `memory_records` rows.
